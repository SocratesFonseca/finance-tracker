# Personal Finance Tracker

## Overview
### This finance tracker solves the problem of losing track of your money. By allowing for categories in expenditures and income, users can target specific issues in unhealthy spending. 

## Highlights
- Transactions can be recurring (weekly, biweekly, monthly, etc.)
- Summary of transactions including income, expenses, and current balance 
- Local CSV storage
- Authentication
- AI financial recommendations

##Tech Stack
### Frontend - React
### Backend - FastAPI (Python)
### Database - PostgreSQL
### AI - Groq API
### Containerization - Docker

## To run:
### Install Docker, get a Groq API key, and clone the repo
### Run "cp .env.example .env"
### Run "docker compose up --build"
### In a browser, run " http://localhost:3000"
